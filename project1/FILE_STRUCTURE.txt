SETUP STEPS
===========
1. Navigate to: c:\visual studio\project1

2. Option A - Automatic (Windows):
   > run.bat

3. Option B - Automatic (Mac/Linux):
   $ chmod +x run.sh
   $ ./run.sh

4. Option C - Manual:
   $ cd backend
   $ python -m venv venv
   $ venv\Scripts\activate  (Windows)
   $ source venv/bin/activate (Mac/Linux)
   $ pip install -r requirements.txt
   $ python app.py

5. Visit: http://localhost:5000

GRID VISUALIZATION
==================
The planner generates a 10x10 grid with:

Zone Colors:
  Gray       = Roads & Transportation
  Yellow     = Residential
  Red        = Commercial
  Green      = Parks & Green Space
  Dark Gray  = Industrial

Zone Distribution:
  40% - Roads (main + secondary)
  25% - Residential
  15% - Commercial
  15% - Parks
  5%  - Industrial (if high density)
